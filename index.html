Small Sample Of Features I'm Proud Of

goaheadtours.com & efultimatebreak.com

Conversion Tools: "Trip Builder"

We iterated time, time and time again on this key conversion tool for us, each time me diving deep into the data and user journeys, and each time coordinating with engineers to understand how far we can push our booking API for more and more personalization.

Achieved 2+ points of conversion per iteration.

Product Discovery: "Tour Match"

I absolutely LOVED this feature. My brain child from start to finish, we solved the problem of trying to match users to one of more than 200 unique products, many of which were very, very similar to new customers. Plugged into multiple systems and required a full stakeholder alignment across sales, marketing, engineering and inventory.

Increased conversions for users of this tool by 10x.

Product Discovery: Deals Filters

A case of experimentation design and iteration, the features on these product discovery deals pages were underperforming like we would like. I worked to isolate variables from within the data and ran A/B and multivariate tests using Optimizely to test towards really wonderful enhancements to the user experience.

Go Ahead Tours Mobile App

I was a key part of the launch team for a best-in-class B2C travel app, including stakeholder alignment, research, data architecture and post-launch documentation, training, monitoring and iteration.

Continued user behavior monitoring and rolling iterations were amongst the rewarding parts of my career!
