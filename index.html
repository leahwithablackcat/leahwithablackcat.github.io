Conversion Tools: "Trip Builder"

We iterated time, time and time again on this key conversion tool for us, each time me diving deep into the data and user journeys, and each time coordinating with engineers to understand how far we can push our booking API for more and more personalization.

2+ Points of conversion per iteration

Product Discovery: "Tour Match"

I absolutely LOVED this feature. My brain child from start to finish, we solved the problem of trying to match users to one of more than 200 unique products, many of which were very, very similar to new customers. Plugged into multiple systems and required a full stakeholder alignment across sales, marketing, engineering and inventory.

10x Increased conversions for users

Product Discovery: Deals Filters

I worked to isolate variables from within the data and ran A/B and multivariate tests using Optimizely to test towards really wonderful enhancements to the user experience.
